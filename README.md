![monokai](./monokai.png)

### Installation guide

```
$ git clone git@github.com:degterev/MonokaiTwist.git
$ cd MonokaiTwist
$ mkdir -p ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
$ cp Monokai\ Revisited.xccolortheme ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
```
